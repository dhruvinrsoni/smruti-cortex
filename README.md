# SmritiCortex

SmritiCortex is the “Everything for the Browser” — an ultra-fast, intelligent search engine that indexes your entire browsing memory and retrieves any URL instantly.

The project name combines:
- **Smriti (स्मृति)**: Sanskrit for “memory”
- **Cortex**: The human brain’s intelligence and memory-recall center

This repository will contain the complete Manifest V3 extension, including:
- A powerful history indexing pipeline
- Modular, pluggable ranking algorithms
- AI-ready search architecture
- Minimal, blazing-fast UI
- Fully documented build & deployment workflows
