    Advanced Browser Commands

    Turns on ~45 power commands (tab/window management, tab groups, browsing data cleanup, Top Sites, and more). Off by default. When you enable this, Chrome will prompt for optional permissions: tab groups, browsing data, and top sites. If you deny the prompt, Advanced Browser Commands stay off. This toggle does not request broad website access (<all_urls>); that is separate and used only for other optional features such as favicons.

    📋 Quick Setup (click to expand)

    Step 1: Install Ollama from ollama.ai

    Step 2: Pull required models:

    # AI keyword expansion (recommended):
ollama pull llama3.2:1b

# Semantic search (embeddings):
ollama pull nomic-embed-text:latest

# Enable CORS (required):
# Windows: setx OLLAMA_ORIGINS "*"
# Mac/Linux: export OLLAMA_ORIGINS="*"
# Then restart Ollama

    Step 3: Enable features below and click 🔄 to verify connection
