    📋 Quick Setup (click to expand)

    Step 1: Install Ollama from ollama.ai

    Step 2: Pull required models:

    # AI keyword expansion (recommended):
ollama pull llama3.2:1b

# Semantic search (embeddings):
ollama pull nomic-embed-text:latest

# Enable CORS (required):
# Windows: setx OLLAMA_ORIGINS "*"
# Mac/Linux: export OLLAMA_ORIGINS="*"
# Then restart Ollama

    Step 3: Enable features below and click 🔄 to verify connection
